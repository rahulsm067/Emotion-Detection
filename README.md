# Emotion Detection in Text

## 📌 Overview
Multi-model NLP system classifying text into 8 emotions using Logistic Regression, SVM, and LSTM.

## ✨ Features
- Three ML models (LR, SVM, LSTM)
- Text & voice input
- Real-time visualization
- Usage analytics dashboard
